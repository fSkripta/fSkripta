Použité symboly
###############

.. glossary::

  |c|
    rychlost proudění (jako |c| se zavádí především v dynamice plynů)

  |cp|
    měrná tepelná kapacita při konstantním tlaku

  Gr
    Grashofovo číslo: parametr vyjadřující poměr vztlaku a vazkých sil.

    .. math::

      \mathrm {Gr}  =\frac {g\beta (T_{w}-T_{\infty })D^{3}}{\nu ^{2}}  
    
 
  |h|
    entalpie
    
    V ideálním plynu platí: :math:`h = 2c_p\ T`

  Nu
    Nusseltovo číslo: parametr vyjadřující vztah mezi konvektivním a konduktivním přenosem tepla. Tím popisuje jejich význam v daném ději.

    .. math::

      \mathrm{Nu} =\frac {\alpha \ L}{\lambda }
  

  Pr
    Prandtlovo číslo: parametr vyjadřující podobnost mezi rychlostním a teplotním polem

    .. math::

      \mathrm {Pr}  = \frac {c_{p}  \ \mu }{\lambda }   

    .. note:: Stojí za povšimnutí, že Prandtlovo číslo je závislé pouze na vlastnostech tekutiny, je tedy také její vlastností a je nezávislé na ději.
      

  Ra
    Rayleigho číslo

    .. math::

      \mathrm {Ra}  = \frac {\rho \beta (T_{w}-T_{\infty }) D^{3}g}{\mu  \alpha } = \mathrm {Gr} \mathrm {Pr}  
  
  Re
    Reynoldsovo číslo: jediný parametr bezrozměrných Navierových-Stokesových rovnic

    .. math::

      \mathrm{Re} =\frac{u \ D}{\nu}  
  
  Sr
    Strouhalovo číslo

    .. math::

      \mathrm {Sr}  =\frac {f \ L}{u}
